## How to get the Microsoft Partner logo

1. Log in to Microsoft Partner Center: https://partner.microsoft.com
2. Go to Membership → Benefits → Logos
3. Download the "Microsoft Partner" badge/logo PNG
4. Save it as: static/images/microsoft-partner.png

The logo is displayed in the footer of seekingspringfield.com.
